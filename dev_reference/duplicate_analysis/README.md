# Duplicate analysis reference files

Questi file sono materiale di riferimento per sviluppare l'analisi duplicati integrata.

- `policy_v12.json`: specifica normativa da tradurre in codice Python interno.
- `sample_inventory.xlsx`: esempio reale di input inventario.
- `expected_duplicate_report.xlsx`: riferimento solo per struttura fogli/colonne/stile motivazioni.

Vincoli:
- Nessun caricamento policy JSON a runtime lato utente.
- Nessun hardcode su titoli/ID/path.
- Nessun copia-incolla delle decisioni dal report atteso.
